Swift Installation on Windows

Try the ways to install Swift on Windows listed on this website. If those do not work for you, try the following step by step instructions instead.

1. Developer Mode

Enable developer mode on Windows by pressing the Windows key and typing developer settings.

1a. Install App Installer (Optional)

You may need to install the Windows App Installer, which is a package manager for Windows. Most likely it is already installed, but if it isn't, please install it.

2. Install Required Dependencies

Open a command prompt by pressing the Windows key and typing cmd. The press the enter key.

In the command prompt, copy and paste each of the following commands, one by one. To paste into the command prompt window, simply click your right mouse button, then hit the enter key.

NOTE: if the winget command doesn't work, it means you need to install the Windows App Installer. 

winget install Git.Git
winget install Python.Python.3.10
curl -sOL https://aka.ms/vs/16/release/vs_community.exe
start /w vs_community.exe --passive --wait --norestart --nocache ^
    --installPath "%ProgramFiles(x86)%\Microsoft Visual Studio\2019\Community" ^
    --add Microsoft.VisualStudio.Component.Windows10SDK.19041 ^
    --add Microsoft.VisualStudio.Component.VC.Tools.x86.x64
del /q vs_community.exe

Here is an example of what it should look like when you copy and paste a command:

Close the command prompt and then open a new command prompt. Run the following command in the new command prompt window: 

pip install six

3. Install the Swift Toolchain

In the same command prompt window, run the following command to install the Swift toolchain: 

winget install Swift.Toolchain

4. Check installation

Close the command prompt and open Visual Studio Code. Open a new terminal in VCSCode and run the following command: 

swift --version
